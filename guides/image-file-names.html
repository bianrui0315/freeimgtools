Naming Images for SEO

Updated July 9, 2026 · A practical image naming convention with filename examples for blogs, ecommerce, galleries, and websites.

Quick answer: Name images for SEO with short, descriptive, lowercase file names separated by hyphens. Use words that describe what is visible in the image, such as blue-linen-shirt-front.webp, and include a keyword only when it naturally fits the image and page topic.

Why image file names matter

A file name is one of the simplest image SEO signals. Search engines read the image URL, and a descriptive file name gives context before alt text or surrounding content is even considered. A name like blue-linen-shirt-front.webp communicates far more than IMG_3021.jpg.

The basic rules

  • Use lowercase letters.
  • Separate words with hyphens, not underscores.
  • Describe what is visible in the image.
  • Include a keyword only when it naturally matches the image.
  • Keep the name short enough to scan.
  • Avoid repeated keywords, random numbers, and camera defaults.

A five-step SEO image naming checklist

  1. Identify the main subject that is actually visible.
  2. Add one useful detail such as color, angle, location, or action.
  3. Use lowercase words separated by hyphens.
  4. Remove filler words, camera numbers, dates, and repeated keywords.
  5. Keep the correct extension, such as .jpg, .png, or .webp.

For example, rename IMG_4402.jpg to white-running-shoes-side-view.jpg when that accurately describes the image. The filename should make sense to a person before they open it.

SEO image naming convention

A simple image naming convention is primary-subject-detail-context.format. For products, use brand-product-color-angle.webp. For guides or blog images, use topic-specific-subject.jpg. The goal is to make the filename useful before someone sees the image.

Good image naming is not keyword stuffing. A name like running-shoes-white-side-view.webp is useful. A name like best-running-shoes-cheap-shoes-buy-running-shoes.webp is spammy and harder to read.

Weak: IMG_4402.jpg, screenshot-final.png, best-shoes-buy-shoes-cheap-shoes.jpg

Better: white-running-shoes-side-view.webp, homepage-performance-audit.png, ceramic-coffee-mug-blue.jpg

Should you rename old images?

If the old image is important and already indexed, rename carefully. Update the page reference, update any sitemap references, and redirect the old image URL when possible. If the image is not important or not indexed, simply improving future filenames may be enough.

FAQ

How should I name images for SEO?

Use short, descriptive, lowercase names with hyphens between words. Describe the image first, then add a keyword only if it naturally matches.

Are hyphens better than underscores?

Yes. Hyphens are the safest separator for SEO-friendly image file names. Avoid spaces and underscores.

Should every image filename include a keyword?

No. Use keywords where they accurately describe the image. Avoid repeated or unrelated keywords.

Should I rename old images?

Rename important old images only if you can update references and redirect old image URLs. For many sites, improving future filenames is safer.
