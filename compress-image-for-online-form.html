Before you submit

Online forms often reject images for two different reasons: file size and pixel dimensions. If the error only mentions KB or MB, compression is enough. If the form mentions exact pixels, use the image resizer first, then compress the resized image.

For identity photos, keep a copy of the original file. Compressing too aggressively can make faces, text, or document edges blurry. If the smallest result is still above the target, reduce the image dimensions and try again.
