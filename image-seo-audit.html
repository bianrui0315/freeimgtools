Enter a website URL and FreeImgTools will scan the page for image SEO problems: missing alt text, weak filenames, missing dimensions, oversized image files, responsive image hints, loading hints, and legacy formats.

Fast URL-based image audit

The scanner fetches the public HTML for one page, checks image tags, and lightly probes a limited number of public image headers for file size. It does not crawl private networks, run JavaScript, upload your files, or measure real Core Web Vitals.

    What this website image scanner checks

    Alt text: Missing alt attributes are accessibility issues and weak image SEO signals. Empty alt text is valid only for decorative images.

    Dimensions: Images without width and height can contribute to layout shift while the page loads.

    Size, filenames, and formats: Descriptive SEO image file names help search engines understand context. Smaller WebP and AVIF files can improve performance compared with oversized JPG or PNG files.

    Responsive delivery: Missing srcset, missing loading hints, and large image headers are surfaced as practical next steps for website owners.
