Best Instagram profile picture size

Instagram profile pictures should be square. A 320 x 320 image is a common display target, while 1080 x 1080 gives you a cleaner source image before Instagram compresses it.

How to resize an Instagram profile picture

  1. Upload a photo, logo, or avatar.
  2. Choose a square size and cover mode.
  3. Download the resized profile picture.

Profile picture tips

  • Keep faces and logos centered.
  • Leave padding around the edges because Instagram displays a circle.
  • Use PNG for logos and JPG for photos.
