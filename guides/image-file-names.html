SEO Image File Names: How to Name Images for Search

Published June 2026 · Practical naming patterns for blogs, ecommerce, galleries, and websites.

Why image file names matter

A file name is one of the simplest image SEO signals. Search engines read the image URL, and a descriptive file name gives context before alt text or surrounding content is even considered. A name like blue-linen-shirt-front.webp communicates far more than IMG_3021.jpg.

The basic rules

  • Use lowercase letters.
  • Separate words with hyphens, not underscores.
  • Describe what is visible in the image.
  • Include a keyword only when it naturally matches the image.
  • Keep the name short enough to scan.
  • Avoid repeated keywords, random numbers, and camera defaults.

Weak: IMG_4402.jpg, screenshot-final.png, best-shoes-buy-shoes-cheap-shoes.jpg

Better: white-running-shoes-side-view.webp, homepage-performance-audit.png, ceramic-coffee-mug-blue.jpg

Naming patterns by content type

  • Product: brand-product-color-angle.webp
  • Blog photo: topic-subject-location.jpg
  • Screenshot: tool-panel-action.png
  • Portfolio: project-name-deliverable-view.jpg
  • Chart: metric-time-period-chart.png

Should you rename old images?

If the old image is important and already indexed, rename carefully. Update the page reference, update any sitemap references, and redirect the old image URL when possible. If the image is not important or not indexed, simply improving future filenames may be enough.
