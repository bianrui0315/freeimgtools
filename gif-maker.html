How to make a GIF from images

  1. Add two or more images in the order you want them to appear.
  2. Choose the GIF size, frame delay, and whether to crop, contain, or stretch frames.
  3. Click Create GIF and download the finished animated GIF.

GIF frame delay and size tips

Frame delay: 500 ms shows two frames per second. Use 100-200 ms for fast motion or 800-1500 ms for a slideshow-style GIF.

Output size: Smaller pixel dimensions create a much smaller GIF. Start around 640 pixels wide for sharing, then reduce the width if the file is too large.

Fit mode: Cover fills the entire frame by cropping edges, contain keeps the full image with empty space, and stretch forces every image to the exact dimensions.

GIF maker FAQ

How do I make a GIF from photos?

Add at least two photos, set the frame delay and output dimensions, then click Create GIF. Your photos are processed in their displayed order.

Can I create a GIF from PNG or WebP images?

Yes. The tool accepts common browser-supported formats including JPG, PNG, WebP, GIF, and BMP.

Why is my GIF file large?

GIF stores every animation frame. Reduce the output width and height, use fewer frames, or increase the encoder quality number for a faster, smaller result.
